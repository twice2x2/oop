Какие сообщения будут выведены при выполнении программы? Перечислите в порядке вызова.
- Default
- Copy
- Move
- Default
- Dtor\n


Объясните, почему вызывается именно та или иная версия конструктора в каждой строке.
- Первая задана с без параметров, то есть по умолчанию
- Вторая задана как b=a, значит копируется значение из а
- Третья задана через move, значит это перемещение
- Четвертая задана тоже без параметров, то есть по умолчанию
- Пятая задана через какой то метод, который обозначает “назад”. Можно предлоложить что это обращение к деструктору


Что произойдёт, если убрать noexcept из конструктора перемещения? (Подсказка: поведение std::vector при реаллокации.)
